# Micro-SaaS Project

## Project Overview
This is a collection of micro-SaaS tools that are:
- Small enough to build solo
- Niche enough to sell  
- Useful enough people will actually pay

## Available Tools Ideas

### 1. D&D Campaign Memory Engine
- Track NPC relationships, plot threads, player decisions
- AI session summaries and lore consistency
- Export session recap PDFs

### 2. AI Prompt Version Manager
- Save and compare prompt versions
- Track cost per prompt
- A/B testing for prompts

### 3. Website Screenshot Change Tracker
- Monitor websites for visual changes
- Automatic screenshots with difference highlighting
- Slack/email alerts

### 4. PDF → Structured Data Extractor
- Extract structured data from PDFs (invoices, statements, etc.)
- Output to JSON/CSV/Excel
- AI fallback for messy layouts

### 5. Etsy Listing Analyzer
- Analyze SEO for Etsy listings
- Keyword recommendations
- Competitor analysis

### 6. Podcast Content Repurposer
- Generate blog posts, tweets, captions from podcast episodes
- Automatic timestamps and chapter markers

### 7. Local Business Review Intelligence
- Analyze Google reviews for sentiment and themes
- Competitor comparison

### 8. Meeting Chaos Organizer
- Extract action items from Zoom transcripts
- Send tasks to Notion/Trello

### 9. Freelance Pricing Calculator
- Calculate rates based on skill, location, experience
- Generate proposals

### 10. Website UX Friction Scanner
- Crawl sites for UX issues
- Generate UX score and recommendations

## Tech Stack Options
- **Python**: Great for AI/ML, data processing, PDF tools
- **Node.js/TypeScript**: Good for web scraping, real-time tools
- **Go**: Excellent for APIs, performance tools
- **PHP**: Simple CRUD, quick deployment

## Getting Started
1. Choose a tool to build
2. Set up the appropriate tech stack
3. Follow the project structure in each tool's folder
4. Focus on MVP features first

## Business Strategy
1. Build component/tool first
2. Sell to developers
3. Turn into micro-SaaS with recurring revenue
